# 🎠 Carousel 프로젝트

### 🔥 Carousel를 바닐라JS를 이용해서 구현하였습니다.

1. default Carousel
2. infinite Carousel
   - 2-1 예제1 overflow unset Carousel
   - 2-2 예제2 반응형 Carousel
   - 2-3 예제3 3D Carousel

<br>

> #### ⚙ 프로젝트 실행 해보기

```js
 git clone https://github.com/dnr14/carousel-vanilaJS.git
 cd carousel-vanilaJS
 npm -g i serve
 serve -s ./보고싶은 프로젝트 경로
```
